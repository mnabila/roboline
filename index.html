ROBOLINE CONTEST 2019

STKIP PGRI Tulungagung

Roboline Contest 2019 adalah sebuah event kontes robot line follower nasional tingkat SMP / MTS / SMA / SMK / MA Se-Indonesia yang di selenggarakan oleh STKIP PGRI Tulungagung. Event Roboline Contest ini diadakan oleh Progam Pendidikan Teknologi Informasi di setiap tahunnya. Pada event ini para peserta akan memperebutkan piala bergilir Gubernur Jawa Timur dan Hadiah Jutaan Rupiah.

Dengan kategori nya antara lain sebagai berikut :

  • Roboline Speed ( Analog )
  • Roboline Speed ( Digital )
  • Roboline Fire Extinguisher ( Digital )
  • RoboFire Extinguisher ( Digital )

  • Untuk pendaftar online waktu maksimal dari pengumpulan semua data adalah 3 x 24 jam (3 hari) setelah transfer biaya pendaftaran
  • Peserta membawa hardcopy dari data 1,2,3,4 dan 5 waktu TM (Technical Metting)

Pendaftaran Via Offline

Pendaftaran offline dilaksanakan secara tatap muka (langsung) dengan panitia, artinya proses pembayaran dan pengumpulan data akan diserahkan ke panitia secara langsung. Langkah-langkah dari pendaftaran via offline adalah :

  1. Mengisi formulir pendaftaran (Formulir bisa di dapat dari panitia / mendownload di www.roboline.stkippgritulungagung.ac.id)
  2. Melakukan pembayaran sesuai dengan nominal yang telah ditetapkan.
  3. Menyiapkan foto 3x4 berwarna, resmi sebanyak 4 lembar (berseragam sekolah / berkerah) untuk masing-masing anggota.
  4. Menyiapkan foto copy pelajar asli sebanyak 2 lembar untuk masing-masing anggota.
  5. Menyiapkan surat pengantar dari sekolah.
Ketentuan untuk pendaftar offline :
  1. Peserta mengirimkan softcopy dari data 3,4 dan 5 dalam bentuk .zip/.rar dengan nama file NamaKetuaTim_Kategori_AsalSekolah (contoh AnwarAnas_RoboSpeed_SMKN1Jakarta) ke e-mail : robolineconteststkipta@gmail.com dengan subject OFFLINE RLC 2019
  2. Peserta membawa hardcopy dari data 3,4 dan 5 waktu TM (Technical Metting)
  3. Waktu maksimal dari pengiriman semua data adalah 3x24 jam (3 Hari) setelah proses pembayaran biaya pendaftaran
